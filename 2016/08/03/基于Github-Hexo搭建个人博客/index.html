基于Github & Hexo搭建个人博客

发表于 2016-08-03   |   分类于 Hexo

简介

我们的目标是：简单 快速 地搭建一个 个人博客。

因此，采用 Hexo 这个东西，它是一个基于nodejs 的静态博客网站生成器，具有：

  • 支持Markdown语法
  • 快速生成静态博客页面文件
  • 丰富漂亮可个性定制的主题
  • 一个指令即可部署到Github免费托管和访问

nodejs 是个什么鬼？

Node.js是一个Javascript运行环境。

呃，就像jre是Java运行环境似的。好吧，没学过，听不懂啊暂时先不搭理它，只要知道Hexo得依赖它才能运行就可以了。

Github，一个项目版本控制工具和代码托管网站。

准备

下载并安装nodejs

去 nodejs官网下载页 选择合适的版本，如我下载的版本是：node-v4.4.7-x64.msi，一路 next 安装即可。

注意： windows系统下，安装nodejs时，要选择把nodejs的命令添加到系统path中。如下：

安装Hexo

打开windows powershell，依次输入：

node -v                  # 查看是否node是否安装成功及其版本

npm install -g hexo     # 安装hexo

hexo version            # 查看hexo是否安装成功，及版本

本地搭建博客

初始化博客

打开windows powershell，在任意地方创建目录 blog-demo，用于存放博客文件。依次执行：

mkdir blog-demo        # 创建目录blog-demo
cd blog-demo        # 进入blog-demo文件夹
hexo init            # 初始化hexo
hexo generate        # 生成静态站点
hexo server            # 启动Hexo

在浏览器中输入：localhost:4000 即可访问默认的 Hexo 博客站点。如下：

发布新的文章

执行命令：

hexo new 文章名称    #发布新文章

根据提示信息，打开创建的md文件，然后编辑完成即可：
