Hexo基本命令和标签

发表于 2016-08-04   |   分类于 Hexo

命令

安装Hexo

npm install hexo -g

初始化

hexo init

生成站点资源

hexo generate

启动服务器

hexo server

发布新文章

hexo new 文章名称

创建新的页面

hexo new page 页面名称

清理已发布站点

hexo clean

部署

hexo deploy
